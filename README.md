Pebble Text Editor
==================
Simple JavaScript library for inline editing.

Features
--------
- Paste plain text
- Bold/Italic
- Make Web Link/Email Link
- Text Color
- Paragraph styles


Support
-------

### Desktop
- Internet Explorer 9+
- Chrome
- Firefox
- Safari
- Opera

### Mobile
- WP Internet Explorer 10+
- iOS Safari 5+
- iOS/Android Chrome
- Android Firefox
